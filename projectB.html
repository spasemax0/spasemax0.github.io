Raspberry Pi Pico Macro Keyboard

Using a raspberry pi pico microcontroller, anyone can make their own keyboard! The amount of buttons is entirely up to the creator as the process to make the first 4 is the same as adding more, this version uses 4 buttons, all programmable in circuit-python that one can bind specific keys or shortcuts too! Personally, I use it for things like opening task manager or media playback/volume control. Overall a customizable, neat and super affordable (probably roughly 20$) gadget that can be used for a plethora of tasks!
